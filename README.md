# Aranybúza pékség


# Technológia:
- React, Vite

# Könyvtár:
- Tailwind
- Iconify
- Framer motion

# Telepítés:

- Előfeltételek
- Node.js v18.0.0 vagy újabb

# Telepítési lépések

A projekt klónozása:
```
git clone https://github.com/dajkagabi/bread.git
```

A projekt könyvtárba lépés:
```
cd bread
```

A függőségek telepítése:
```
npm install
```

A fejlesztői szerver indítása:
```
npm run dev
```
